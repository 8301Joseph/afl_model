How predictions are calculated

Overview

Every prediction combines three independently-tracked rating systems — Elo, offensive ratings, and defensive ratings — into a Ridge regression model trained on historical AFL data. All three systems update after every game, so predictions adapt to current form throughout the season.

Elo Ratings Core signal

A classic chess-style rating system adapted for AFL. Each team starts with a rating of 1500 and gains or loses points based on who they beat and by how much.

Offensive & Defensive Ratings Form signal

Each team tracks how many points they score (offensive) and concede (defensive) using an exponentially-weighted moving average — so recent games matter more than older ones.

  • Off rating — average points scored per game, weighted toward recent form
  • Def rating — average points conceded (lower is better)
  • New teams start at the league average (~88 pts)
  • The smoothing factor (α = 0.18) means ~5 recent games have the most influence

Regression Model Combines everything

A Ridge regression is trained on all historical games to predict the final margin (home score − away score). Features fed into the model:

  • Elo diff — home Elo minus away Elo
  • Off diff — home offensive rating minus away offensive rating
  • Def diff — away defensive rating minus home defensive rating
  • Home advantage — per-team home ground boost, learned from historical margins at each venue

Recent games are weighted more heavily using exponential time decay — games from 2+ seasons ago have roughly 25% of the weight of a current game.

Win Probability & Predicted Margin

The model outputs a predicted margin. Win probability is derived from that margin using a normal distribution fitted to historical margin variance — so a larger predicted margin means a higher win probability.

Projected Ladder

The ladder shows expected wins (xW) — not actual wins. For every remaining game, the model calculates each team's win probability. These probabilities are summed across all remaining games to project how many wins each team is likely to finish with. Actual results already played are included as full wins or losses.

Reading the Cards

  • The bar shows the home/away win split (e.g. 65% / 35%)
  • Blue name = model's tipped winner (upcoming games)
  • Green name = actual winner (past games)
  • Red name = team the model tipped but lost (past games)
  • Past cards show the tipped winner + predicted margin vs the actual result
